# 多语言适配（QTranslator）

## 方案

使用 Qt 官方 QTranslator 机制：代码中 `self.tr()` 包裹字符串 → `pyside6-lupdate` 扫描生成 `.ts` → Qt Linguist 翻译 → `pyside6-rcc` 编译 `.qm` → 运行时 `QTranslator.load()` 加载。

## 开发步骤

### Step 1：GUI 类 `self.tr()` 替换

对每个 QWidget 子类的 `.py` 文件：
- 所有用户可见字符串 `"..."` → `self.tr("...")`
- `f"..."` 内的变量 → `self.tr("...").arg(var)`
- `QMessageBox.question(self, "标题", "内容")` → `self.tr("标题"), self.tr("内容")`

批量替换规则：

| 原始 | 替换为 |
|------|--------|
| `"扫描本机已装软件"` | `self.tr("扫描本机已装软件")` |
| `f"发现 {n} 个文件"` | `self.tr("发现 %1 个文件").arg(n)` |
| `f"[{i+1}/{t}] 正在备份 {n}..."` | `self.tr("[%1/%2] 正在备份 %3...").arg(i+1).arg(t).arg(n)` |

涉及文件：
- [ ] `src/gui/main_window.py` — 窗口标题 + Tab 标签
- [ ] `src/gui/home_tab.py` — 按钮文字、标签、状态文本、消息框、表格表头
- [ ] `src/gui/config_tab.py` — Tab 标签、按钮、验证标签、消息框
- [ ] `src/gui/config_wizard.py` — 步骤名、按钮、标签、对话框标题、表单字段名
- [ ] `src/gui/settings_tab.py` — 组框标题、标签、选项、关于信息

### Step 2：非 QWidget 类翻译

非 QWidget 类没有 `self.tr()`，用 `QCoreApplication.translate(context, source)`：

```python
from PySide6.QtCore import QCoreApplication

def _tr(text: str) -> str:
    return QCoreApplication.translate("BackupEngine", text)
```

涉及文件：
- [ ] `src/core/backup_engine.py` — `self.signals.message.emit(...)` 中的文字
- [ ] `src/core/restore_engine.py` — `self.signals.message.emit(...)` 中的文字

### Step 3：生成 .ts 文件

```powershell
pyside6-lupdate src/ src/gui/ src/core/ src/storage/ src/utils/ main.py -ts lang/zh_CN.ts
```

首次：
- [ ] 创建 `lang/` 目录
- [ ] 生成 `lang/zh_CN.ts`（中文源）
- [ ] 复制为 `lang/en_US.ts`（翻译目标）
- [ ] 写 `lang/update.bat` 一键更新脚本

### Step 4：翻译（Qt Linguist 或手写 XML）

用 Qt Linguist 打开 `en_US.ts`，逐条翻译。或直接在 XML 中填写 `<translation>` 标签。

- [ ] 翻译英文 `en_US.ts`（~130 条）
- [ ] 后续可按需添加 `ja_JP.ts`、`ko_KR.ts` 等

### Step 5：编译 .qm

```powershell
pyside6-lrelease lang/en_US.ts -qm lang/en_US.qm
```

- [ ] 编写 `lang/release.bat` 一键编译所有 .ts → .qm
