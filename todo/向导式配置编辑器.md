# 向导式配置编辑器

## 设计目标

在现有 JSONC 源码编辑模式之外，增加一个可视化 Step-by-Step 配置向导，降低用户编写配置规则的门槛。两模式共存，用户通过按钮切换。

## 模式切换

规则管理器右侧由 QTabWidget 切换两种模式：

```
┌───────────────────────────────┐
│ [源码模式] [向导模式]    [保存] │
├───────────────────────────────┤
│  ┌─ 向导步骤 ──────────────┐  │
│  │ ● 基本信息              │  │
│  │ ○ 源路径                │  │
│  │ ○ 解析字段              │  │
│  │ ○ 备份策略              │  │
│  └─────────────────────────┘  │
│                               │
│  ┌─ 当前步骤内容 ──────────┐  │
│  │  (表单字段)              │  │
│  │                          │  │
│  └──────────────────────────┘  │
│                               │
│  [上一步] [下一步] [完成]     │
└───────────────────────────────┘
```

## 步骤设计

### Step 1：基本信息

| 字段 | 控件 | 说明 |
|------|------|------|
| `name` | QLineEdit | 必填，规则显示名称 |
| `description` | QLineEdit | 选填，鼠标悬停提示 |
| `platform` | QComboBox | windows / macos / linux / cross-platform |
| `enabled` | QCheckBox | 默认勾选 |

### Step 2：源路径

```
┌─ 源路径列表 ──────────────────────┐
│ %APPDATA%\Code\User\settings.json │
│ %APPDATA%\Code\User\keybindings.. │
│ %APPDATA%\Code\User\snippets\     │
└────────────────────────────────────┘
[添加路径] [删除路径] [浏览...]
```

- 添加路径：弹出 QInputDialog 手写，或 QFileDialog 选择文件/目录后自动填入
- 支持 `%xxx%` 环境变量，浏览时自动转换

### Step 3：解析字段

```
┌─ 关键字段 ────┬──────┐
│ editor.fontSize │ 字号  │
│ editor.fontFamily│ 字体  │
│ theme           │ 主题  │
└────────────────┴──────┘
[添加字段] [删除字段]
```

- 弹出对话框输入：字段路径（点号格式）+ 显示标签
- 校验路径格式（不允许数组索引）

### Step 4：备份策略

| 字段 | 控件 | 说明 |
|------|------|------|
| `strategy.type` | QComboBox | 增量备份 / 全量备份 |
| `strategy.max_versions` | QSpinBox | 范围 1-99，默认 10 |
| `strategy.ignore_patterns` | QListWidget + 添加/删除 | Glob 模式列表 |

### 完成

点击「完成」→ 校验必填项 → 将表单数据序列化为 JSONC → 写入文件 → 刷新列表。

## 数据流

```
向导表单数据 (dict)
    ↓ serialize_to_jsonc()
符合格式的 JSONC 字符串
    ↓ write_text()
配置文件 (.jsonc)
    ↓ json5.loads()
反向填充向导各步骤 (parse_to_form_data)
```
